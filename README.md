# 航空宇宙工学実験「宇宙機の姿勢決定」

九州大学・航空宇宙工学実験のテーマ「宇宙機の姿勢決定」の配布リポジトリです。
軌道上サービス（OOS）を想定し、**固有空間法（PCA → 射影 → 最近傍）**でターゲット宇宙機の相対姿勢を推定します。

## 実験の概要

ロボットアームで宇宙機モデルを単軸スピンさせ、リニアレール上のカメラで撮影した画像から姿勢角を推定します。
本実験では学習データに **CGレンダリング画像（配布済み）** と **実画像** の両方を使い、次の3つの軸で精度を比較します。

1. **CG学習 vs 実画像学習**（同一テスト条件）→ Sim2Real ドメインギャップの定量化
2. **基準条件 vs 条件変更**（同一学習）→ 条件汎化性能の定量化
3. **CG学習 × 条件変更 vs 実画像学習 × 条件変更** → 学習ソースが頑健性に与える影響

実験室では基準条件と条件変更（光源方向を変える）の2セットの実画像評価データを取得します。

## 進め方

1. `docs/main.pdf` の実験手順書を読む（**まずこれ**）
2. `code/controller.mlapp` で実験装置を制御し、画像を撮影
3. `code/eigenSpaceCG.m` で固有空間法を実行し、3軸の精度比較を確認
